# Logo assets

Agent brand marks used by the root README's "Supported agent CLIs" grid.

Sourced from **[Simple Icons](https://simpleicons.org)** (icons are CC0) and
recolored for theme contrast. Each brand's mark/name is a trademark of its owner
and is used here only to indicate supported integrations.

| File(s) | Agent | Simple Icons slug | Fill |
|---|---|---|---|
| `claude.svg` | Claude Code | `claude` | brand `#d97757` (single — reads on both themes) |
| `openai-light.svg` / `openai-dark.svg` | OpenAI Codex | `openai` | `#1f2328` / `#e6edf3` |
| `copilot-light.svg` / `copilot-dark.svg` | GitHub Copilot CLI | `githubcopilot` | `#1f2328` / `#e6edf3` |
| `cursor-light.svg` / `cursor-dark.svg` | Cursor Agent CLI | `cursor` | `#1f2328` / `#e6edf3` |
| `opencode-light.svg` / `opencode-dark.svg` | OpenCode | `opencode` | `#1f2328` / `#e6edf3` |
| `pi-light.svg` / `pi-dark.svg` | Pi (pi-coding-agent) | `pi` | `#1f2328` / `#e6edf3` |
| `gemini-light.svg` / `gemini-dark.svg` | Gemini CLI | `googlegemini` | `#1f2328` / `#e6edf3` |

`*-light.svg` is the dark glyph shown on light backgrounds; `*-dark.svg` is the
light glyph shown on dark backgrounds (the README's `<picture>` switches on
`prefers-color-scheme`).

To refresh: re-fetch `https://cdn.jsdelivr.net/npm/simple-icons/icons/<slug>.svg`
and inject the `fill` above.
